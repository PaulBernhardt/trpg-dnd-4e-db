How to Use

  1. You need an active (paid) D&D Insider subscription.
    You can check / recover your DDI account from its official site. This program cannot access protected content without an active subscription.
  2. Fill the login in Data tab.
    This downloader relies on you to fill in correct DDI login information. This downloader only send your login to the DDI site. Password is remembered on your machine (though not with saved data); delete it after download to be safe.
  3. Click "Download" and leave it running
    The download process takes at least a day or longer. Progress is saved in real time, so it can be stopped mid-way and it will resume from there next time. The download step requires Internet connection and an active subscription.
  4. Click "Export" and save it somewhere
    Once data is complete, the download button will become export button. Export is a process of converting downloaded data into a form that web browsers can read. It starts anew every time, usually takes a few minutes, and need 1GB memory.
  5. (Optional) Transfer the exported file and data to mobile devices
    The exported .html file can be opened in any modern browser, and it comes with a same named folder that contains the data. You can copy the file and the folder to other PC or mobile devices; copying data to your personal devices for personal use does not violate copyright.
  6. If you find a typo or obvious mistake in the compendium, please file an issue. I cannot update the official compendium, but I can update this downloader's data export.

Troubleshoot

  1. How to get DDI subscription?
    1. You need to have registered and paid before 2015 Dec.
    2. Old email based DDI account must be upgraded, which happens when you login to DDI.
    3. Find help from your D&D community if you never subscripted, sorry I can't help.
    4. To buy access, view an old subscription in order history and click "renew" to place a new order.
    5. Once paid, you can disable auto-renew by viewing the new order and follow its manage link.
  2. Cannot open / read / write database
    Try "Reset Data" in option tab. You will need to re-download the data.
    Data is stored in "dnd4_compendium.database", and backups may be available in same folder.
  3. Connection timeout
    Increase the timeout period in option tab; long pages like power list will take a while to retrieve.
  4. Connection reset It usually means your Internet connection was abruptly broken. Try increase retry count in option tab, or try another network.
  5. Slow download
    Whoever is patient has great understanding, but one who is quick-tempered displays folly. - Proverbs 14:29
    Decrease the throttle rate in option tab, but there are 25k pieces of data to download after all. That is 14 hours if your Internet can get it at 1 per 2 seconds.
  6. How to export to my phone / pad?
    USB drive is pretty fool-proof. Android is also easy. Just remember to transfer the data folder too (Windows's File Explorer will do that for you.)
    Other mobile OS (iOS, ChromeOS, Win mobile etc) I have no clue.
  7. java.nio.channels.ClosedChannelException Only got this error once. Simply close the downloader and re-run.
