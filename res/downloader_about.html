How to Use

  1. You need an active (paid) D&D Insider subscription.
    You can check / recover your DDI account from its official site. This program cannot access protected content without an active subscription.
  2. Fill the login in Data tab.
    This downloader relies on you to fill in correct DDI login information. This downloader only send your login to the DDI site. Password is remembered on your machine (though not with saved data); delete it after download to be safe.
  3. Click "Download" and leave it running
    The download process takes at least a day or longer. Progress is saved in real time, so it can be stopped mid-way and it will resume from there next time. The download step requires Internet connection and an active subscription.
  4. Click "Export" and save it somewhere
    Once data is complete, the download button will become export button. Export is a process of converting downloaded data into a form that web browsers can read. It starts anew every time, usually takes a few minutes, and need 1GB memory.
  5. (Optional) Transfer the exported file and data to mobile devices
    The exported .html file can be opened in most browser, and it comes with a same named folder that contains the data. You can copy the file and the folder to other PC or mobile devices.
  6. If you find a typo or obvious mistake, please file an issue. I cannot update the official compendium, but I can update this downloader's data export.

Troubleshoot

  1. How to get DDI subscription?
    1. You need to have registered and paid before 2015 Dec.
    2. Old email based DDI account must be upgraded, which happens when you login to DDI.
    3. Find help from your D&D community if you never subscripted, sorry I can't help.
    4. To buy access, view an old subscription in order history and click "renew" to place a new order.
    5. Once paid, you can disable auto-renew by viewing the new order and follow its manage link.
  2. Cannot open / read / write database
    Data is stored in "dnd4_compendium.database", and a backup is normally in the same folder which you can try to manually restore.
    If that doesn't help, try "Reset Data" in option tab. You will need to re-download all data after reset.
  3. Connection timeout
    Increase the timeout period in option tab; long pages like power list will take a while to retrieve. Can also mean a broken or blocked Internet connection.
  4. Connection reset
    Try increase retry count in option tab, or try a more stable network.
  5. Slow download
    Whoever is patient has great understanding, but one who is quick-tempered displays folly. - Proverbs 14:29
    Decrease the throttle rate in option tab, but there are 25k pieces of data to download after all. That is 14 hours if your Internet can get it at 1 per 2 seconds.
  6. Slow export
    Temporary disable anti-virus before export. (Windows has a default AV called Windows Defender.) You can also add the export location to the anti-virus' white list.
    Alternatively, get a faster computer. My development PC can go from startup to finish export in 8 seconds.
  7. Slow export at the end
    Disk writes are buffered in system memory and/or harddisk memory. When the write cache is full, subsequence writes has to wait. This effect is particularly pronounced on this program, which may improve in future versions.
  8. ~ failed (script error)
    This means the login form or compendium has been changed (again). May need a programmer to solve this one. See "other errors" below.
  9. Unknown host
    Most likely, wizards has pulled the plug on ddi.wizards.com.
  10. Not enough space
    Check the drive where you are exporting data. Exported data is 60 MB in slze, but some devices may require 100+ MB.
    If you have used old version (3.5.1 or before) and is overwriting older export, please delete old export first. New version use a more efficient format that dramatically reduce waste space and speedup file copy.
  11. Data is different from official compendium
    This downloader will fix known mistakes and re-catogarise some data. See project homepage for a summary, and the in-app debug log will log each affected entry.
  12. Other errors
    For export problem - like a ClosedChannelException that I ran into once - try close program and re-export.
    For download problem, many things can go wrong, such as if Wizards changed something again.
    If you enable debug tabs and restart program, you can see a detailed log, and can also see and control the embedded browser.
    For example, if you determine that there may be a login problem, you can try to login from the browser. That may allows downloads to proceed normally.
    If you wish to report an issue, remember to copy and paste the log or save it as an attachment, perhaps with a screenshot of the internal browser.
