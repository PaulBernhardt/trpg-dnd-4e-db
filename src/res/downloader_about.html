4e Compendium (v3.5) Downloader

Work In Progress. Program Not Working.

How to Use

  1. You need an active (paid) D&D Insider subscription.
    You can recover your DDI account from the official site. Note that DDI no longer accepts new subscription. If you have never subscripted, please find help from your D&D community. Otherwise, view any old subscription in order history and click "renew" to place a new order.
  2. Fill the login in Data tab.
    This downloader relies on you to fill in correct DDI login information. This downloader only send your login to the DDI site. (If it hasn't been modified from the original.)
  3. Click "Download" and leave it running
    The download process takes at least a day or longer. Progress is saved in real time, so it can be stopped mid-way and it will resume from there next time. This is the only step that requires Internet connection and an active subscription.
  4. Click "Export" and save it somewhere
    Once data is complete, the download button will become export button. Export is a process of converting downloaded data into a form that web browsers can read. It starts anew every time, takes at least a few minutes, and eats lots of memory.
  5. (Optional) Transfer the exported file and data to mobile devices
    The exported .html file can be opened in any modern browser, and it comes with a same named folder that contains the data. You can copy the file and the folder to other PC or mobile devices; copying data to your personal devices for personal use does not violate copyright.
